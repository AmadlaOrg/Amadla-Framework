Amadla Documentation
====================

Amadla Framework is the basic structure that you need to make your own cloud and server setup.

To make it work you need to install `amadla-cli <https://github.com/AmadlaOrg/amadla-cli/discussions>`__.

You could say that `amadla-cli <https://github.com/AmadlaOrg/amadla-cli/discussions>`__ is the brains and Amadla Framework is the body.
